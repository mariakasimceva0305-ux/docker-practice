# Docker Practice

Контейнеризированный набор сервисов для вычисления функций потерь с единым запуском через `Docker Compose`.

## Назначение

Разнести несколько вычислительных сценариев по отдельным контейнерам и запускать их в едином окружении.

## Архитектура

```text
Docker Compose
      │
      ├─ сервис MSE
      ├─ сервис MAE
      └─ сервис Squared Hinge Loss
```

## Что реализовано

- отдельные сервисы для вычисления функций потерь;
- единая сборка через `Dockerfile`;
- совместный запуск через `docker-compose.yml`.

## Структура репозитория

- сценарии вычисления функций потерь;
- `Dockerfile`;
- `docker-compose.yml`;
- зависимости Python.

## Реализованные функции

- `Mean Squared Error`
- `Mean Absolute Error`
- `Squared Hinge Loss`

## Запуск

```bash
docker compose up --build
```

## Ограничения

- репозиторий сфокусирован на инфраструктурной упаковке;
- расширение предполагает единый формат входных данных и примеры вызова каждого сервиса.
