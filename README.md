# Docker Practice

Контейнеризированный набор сервисов для вычисления функций потерь и запуска через `Docker Compose`.

## Задача
Нужно разнести несколько вычислительных сценариев по отдельным контейнерам и запускать их в едином окружении.

## Архитектура решения
```text
Docker Compose
      │
      ├──► сервис MSE
      ├──► сервис MAE
      └──► сервис Squared Hinge Loss
```

## Структура репозитория
- сценарии вычисления функций потерь;
- `Dockerfile`;
- `docker-compose.yml`;
- зависимости Python.

## Реализованные функции
- `Mean Squared Error`
- `Mean Absolute Error`
- `Squared Hinge Loss`

## Запуск
```bash
docker compose up --build
```

## Ограничения и развитие
- репозиторий сфокусирован на инфраструктурной упаковке, а не на сравнении моделей;
- следующий шаг — добавить единый интерфейс входных данных и примеры использования для каждого сервиса.
